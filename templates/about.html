About the Project

This project aims to help farmers identify diseases in maize crops using machine learning. By uploading an image of maize leaves, the system can detect whether the crop is affected by Blight, Grey Leaf Spot, Common Rust, or if it is Healthy.

The system uses a ResNet18 convolutional neural network (CNN) trained on a dataset of maize leaf images. It provides quick and accurate predictions to assist farmers in taking timely action to protect their crops.
